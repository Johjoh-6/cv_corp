# CV-Corp
This is only a theme Wordpress of CV Corp

## Instalation
```bash
git clone https://github.com/Johjoh-6/cv_corp
cd cv_corp

npm install
```

For run Webpack. 
```bash
npm run watch
```
or
```bash
npm run build
```

For run PHP
```bash
php -S localhost:3000 -t public
```
